# ANTLRv4-In-Python

## How to use:-

```
python traverse.py <grammar_name> <grammar_rule> <file_path>
```

## Example:-

```
 python traverse.py JSON json sample.json 
 ```
 
 ## Output:-
 
```
AST output:
(json (value (obj { (pair "a" : (value 1)) , (pair "b" : (value (obj { (pair "c" : (value 2)) , (pair "d" : (value 3)) }))) })))

```
